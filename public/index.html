Welcome to the GNU Radio SOAPY Block

gr-Soapy is an OOT module for the GNU Radio platform based on SoapySDR API. The main goal of the blocks are to provide a generic interface to effortlessly configure SDR devices. gr-Soapy implements two distinct blocks, one for RX and one for TX, that dynamically change their parameter fields according to the specified device used to simplify the necessary options to be configured by the user. Also, for devices that are not specifically supported by the blocks, various general options are avaible to configure any device that is supported by the SoapySDR API.
